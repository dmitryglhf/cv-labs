Решить задачу семантической сегментации или object detection на датасете вашего
выбора. Решение должно включать код и небольшой письменный отчет.
Код решения должен включать следующие элементы:
  ●​ создание модели
  ●​ обучение модели
  ●​ использование аугментаций изображений
  ●​ подбор гиперпараметров обучения
  ●​ оценка обобщающей способности итоговой модели.
Отчет должен включать:
●​ постановка задачи
  ○​ описание задачи
  ○​ описание использованных выборок данных
  ○​ описание выбранной метрики качества
  ○​ обоснование выбора метрики качества
●​ модель
  ○​ описание модели (архитектура, способ предобучения (если применимо))
  ○​ блок-схема модели
  ○​ обоснование выбора модели
●​ обучение
  ○​ описание гиперпараметров обучения
    ■​ фиксированные гиперпараметры с обоснованием выбора значений
    ■​ пространство подбираемых гиперпараметров
    ■​ способ подбора гиперпараметров
